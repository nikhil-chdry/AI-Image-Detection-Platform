# 🔍 AI Image Detection Platform + Portfolio Website

A full-stack AI-powered web application that detects 
whether an uploaded image is AI-generated or real,
integrated into a modern portfolio website.

## 🧠 Research Foundation
Inspired by 4 IEEE Research Papers:
- Paper 1: LSTM + Facial Geometric Features (Xiong et al.)
- Paper 2: Prescriptive Ensemble Architecture (Mishra et al.)
- Paper 3: Iterative GAN Detection (Chu)
- Paper 4: Feature Fusion + FGSM Defense (Singh et al.)

## 🛠️ Tech Stack
### Frontend
- React.js + Vite + Tailwind CSS + Framer Motion

### Backend
- Node.js + Express + MongoDB + JWT

### AI/ML
- Python + PyTorch + OpenCV + FastAPI

## 📊 Model Architecture
- ResNet50 + XceptionNet Feature Fusion
- Inspired by 4 IEEE Research Papers
- Trained on CIFAKE + FaceForensics++ dataset

## 📈 Target Results
- Accuracy: ~94%
- F1 Score: ~95%
- AUC: ~96%
